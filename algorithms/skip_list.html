Skip List

A skip list is made up of several layers of linked lists and optimises for faster search times. The first layer - Layer 0 - is the full list of elements. The next layer - Layer 1 - is made up of elements from Layer 0 at the same position as those elements. An element is repeated in Layer 1 from Layer 0 if it's chosen by a probability function. This process is repeated until Layer N ends up with only 1 node.
